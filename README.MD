C4-Code

Software architecture skill for representing system design at the code level using the C4 model.

Overview

Covers the lowest level of the C4 model, focusing on code structure, class relationships, module boundaries, and implementation details within components. Helps bridge high-level architecture with actual source code organization.

Usage

Used to document internal structure of components, improve codebase understanding, and support maintainable system design aligned with architectural intent.
